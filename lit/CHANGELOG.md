# Lattice Identity Token Specification (LIT‑Spec) — Changelog

## v1.0 — Initial Draft Specification
- first publication of the Lattice Identity Token Specification  
- defines identity token format  
- defines identity metadata structures  
- defines identity provenance metadata  
- defines identity constraints  
- defines attestation metadata  
- defines revocation metadata  
- defines identity boundary metadata  
- defines identity validity metadata  
- defines canonical serialization rules  
- defines identity error semantics  
- establishes LIT‑Spec as the technical substrate for identity across the Lattice ecosystem  
- aligned with LIT v1.0, TS‑Spec v1.0, LP‑Spec v1.0, LM‑Spec v1.0, and LW‑Spec v1.0  
- added canonical metadata and publication structure
