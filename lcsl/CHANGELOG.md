# LCSL‑Spec v1.0 — Changelog

## v1.0 — Initial Publication
- First publication of the Lattice Certification & Safety Layer Specification (LCSL‑Spec).
- Defines technical structures for certification metadata, attestation fields, provenance schemas, domain transition structures, and capability declarations.
- Specifies algorithms for version floor enforcement and update sequencing.
- Provides implementation requirements for interoperable, verifiable, and safe LCSL‑conformant systems.
